# Support

## Where to Ask

- **Bug reports:** use GitHub Issues after confirming the problem is reproducible with a current CC Desk release.
- **How-to questions and ideas:** use GitHub Discussions.
- **Security vulnerabilities:** follow [SECURITY.md](SECURITY.md); do not post them publicly.
- **Code contributions:** follow [CONTRIBUTING.md](CONTRIBUTING.md).

## Before Opening a Bug Report

Include the CC Desk version, operating system and architecture, Claude Code version, steps to reproduce, expected and actual behavior, and redacted logs when relevant. Remove API keys, tokens, private paths, project names, session content, and Provider configuration before posting.

## Support Boundary

CC Desk maintainers support official CC Desk releases and documented integrations. We cannot provide general support for Claude Code accounts, third-party Provider billing, MCP servers, third-party plugins, operating-system administration, or recovered user data.
